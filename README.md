# クイズをシェアして、学習をブースト！ 新感覚の英単語アプリ「シェアタン」登場

「英単語学習は、一人で黙々と続けるのがつらい…」 「暗記ばかりで、モチベーションが続かない…」

そんな悩みを抱えるすべての英語学習者へ。 「シェアタン」が、あなたの学習体験を変えます。

「シェアタン」は、豊富な単語帳での学習はもちろん、 学習した単語のクイズを他のユーザーと出題し合える 【 クイズシェア機能 】を搭載した、ソーシャル学習アプリです。

一人では挫折しがちな単語学習を 「みんなで楽しむイベント」に変え、 あなたの英語学習を強力にサポートします。

## シェアタンの主な特徴
1. 【 目玉機能 】クイズシェア機能
〜 一人じゃない、みんなで学ぶ楽しさ 〜

「シェアタン」最大の魅力は、 ユーザー同士でクイズを出し合える「クイズシェア機能」です。

▼ 知識のアウトプット 自分が学習した範囲から、オリジナルの四択クイズを作成し、他のユーザーに出題できます。

▼ 記憶の定着 他人に「教える（出題する）」ことは、最も効果的な学習法の一つ。クイズを作るプロセスで、あなたの理解度も飛躍的に深まります。

▼ 続くモチベーション 友達やライバルとクイズを交換することで、ゲーム感覚で楽しく学習が続けられます。「一人じゃない」という感覚が、あなたの学習意欲を刺激します。

2. 【 習慣化をサポート 】通知Bot機能
〜 忙しくても忘れない 〜

忙しい毎日でも学習を忘れないように、 設定した時刻に単語クイズをお届けする「通知Bot機能」を搭載。

「通学中の10分」 「寝る前の5分」

あなたのライフスタイルに合わせて学習時間をリマインド。 スキマ時間を活用して、無理なく学習を習慣化できます。

3. 【 圧倒的網羅性 】充実の単語帳ラインナップ
〜 このアプリ一つでOK 〜

初学者から上級者、受験生から社会人まで、 あらゆる学習目的に対応する単語帳を標準収録。

==================== 収録単語帳の一例
◆ 基礎 ・NGSL (New General Service List)

◆ 大学受験 ・『ターゲット』・『速単』・『シス単』シリーズの英単語＆英熟語を徹底解析し、入試に必須の単語を網羅！

◆ 社会人向け ・TOEIC頻出単語 ・ビジネス頻出英単語

4. 【 シンプル設計 】効率的な四択クイズ演習
〜 サクサク進めて定着させる 〜

学習の基本は、シンプルな四択クイズです。 ゲーム感覚でサクサクと単語の理解度をチェック。

間違えた問題は効率的に復習できるため、着実に弱点を克服し、語彙力を強化できます。
